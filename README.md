# Password Strength Analyzer

## Overview
The Password Strength Analyzer is a cybersecurity tool written in Python that evaluates password security using rule-based analysis and entropy calculations.

The goal of this project is to demonstrate fundamental authentication security concepts and password attack resistance.

## Features

• Password complexity checks  
• Detection of weak or common passwords  
• Entropy calculation for password strength  
• GUI interface for easy interaction  
• Security recommendations for improving passwords  

## Security Concepts Demonstrated

- Password complexity policies
- Brute-force attack resistance
- Password entropy
- Common password vulnerabilities

These concepts align with recommendations from OWASP password security guidelines.

## Technologies Used

Python  
Tkinter (GUI)  
Regular Expressions  
Entropy calculations
